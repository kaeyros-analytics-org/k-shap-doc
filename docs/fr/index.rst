Documentation de l'application K-Shap de Gestion des Décaissements
====================================================================

.. image:: /images/dashboard.png

Vue d'ensemble
===================

**K-Shap** est une application web conçue pour gérer et surveiller les décaissements au sein d'une organisation. Elle offre des outils pour gérer les étapes de décaissement, contrôler les validations et notifier les validateurs à chaque étape du processus de décaissement. L'application prend en charge l'ensemble du cycle de vie d'un décaissement, en s'assurant que chaque étape est validée selon des critères prédéfinis.
