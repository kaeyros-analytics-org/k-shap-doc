Overview
==========

**K-Shap** is a web application designed for managing and monitoring disbursements within an organization. It provides tools for managing disbursement steps, controlling validations, and notifying validators at every stage of the disbursement process. The application supports the entire lifecycle of a disbursement, ensuring that each step is validated according to predefined criteria.
